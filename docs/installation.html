Hebrew RTL Plugin

Automatic Right-to-Left Text Formatting for Claude Code

What Is This Plugin?

The Hebrew RTL Plugin automatically adds Right-to-Left formatting markers to Hebrew text in your Claude Code editor. When you write in Hebrew, the plugin detects it and ensures proper display direction.

✓ Automatic: Works silently in the background after every write or edit
✓ Smart: Only affects Hebrew text, leaves English untouched
✓ Simple: No configuration needed—just install and forget

Quick Installation

1 Clone the repository 
git clone https://github.com/aviz85/hebrew-rtl-plugin.git
cd hebrew-rtl-plugin
2 Install as a Claude Code plugin 
claude plugin install . --scope user
3 Reload plugins

In Claude Code, run:

/reload-plugins
4 Verify installation

Check that the plugin is active:

/plugin

You should see hebrew-rtl in the list of active plugins.

How It Works

The Smart Detection

The plugin uses intelligent paragraph analysis:

  • Detects the first letter in each paragraph
  • Skips non-letter characters (numbers, punctuation, spaces)
  • Checks if that letter is Hebrew
  • Adds the RTL marker only if Hebrew text is found

Example Transformations

Example 1: Numbered Hebrew Item

Before: 
1. קטגוריה א
After (automatic): 
‫1. קטגוריה א

The RTL marker is added at the very beginning, before the "1".

Example 2: English Stays Unchanged

Before: 
English paragraph with text
After (no change): 
English paragraph with text

English paragraphs are never modified.

Troubleshooting

Plugin not working?

Check these:
  1. Is it installed? Run /plugin to see active plugins
  2. Reload plugins: Run /reload-plugins
  3. Make sure script is executable: chmod +x scripts/hebrew-rtl-fixer.py
  4. Enable debug mode: claude --debug

Only affects some paragraphs?

The plugin correctly identifies Hebrew by the first letter it finds. If a paragraph starts with non-letter characters, the plugin looks for the next letter to determine if it's Hebrew or English.

Manual Installation

If you prefer not to use the CLI:

1. Copy the plugin directory 
cp -r hebrew-rtl-plugin ~/.claude/plugins/hebrew-rtl
2. Make script executable 
chmod +x ~/.claude/plugins/hebrew-rtl/scripts/hebrew-rtl-fixer.py
3. Reload in Claude Code 
/reload-plugins

For Developers

Testing the Plugin

# Run Claude Code with the plugin
claude --plugin-dir ./hebrew-rtl-plugin

# Test the script directly
python3 scripts/hebrew-rtl-fixer.py test-file.md

Modifying the Script

The fixer script is in scripts/hebrew-rtl-fixer.py. It's a standalone Python 3 script that can be tested independently.
